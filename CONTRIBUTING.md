# Contribuer

Merci pour l'intérêt ! Voici comment contribuer rapidement :

1. Fork le projet.
2. Crée une branche : `git checkout -b feat/ma-fonctionnalite`.
3. Fais tes modifications.
4. Ouvre une PR depuis ton fork vers `main`.

Format et bonnes pratiques :
- Respecte le style (indentation 2 espaces).
- Tests/validation : assure-toi que ton changement ne casse pas le rendu global.
- Ajoute une description claire dans la PR.

Voir aussi `.github/ISSUE_TEMPLATE` pour ouvrir une issue.
